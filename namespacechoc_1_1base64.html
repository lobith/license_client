Encodes a chunk of binary data as base-64. The lambda function provided for the output callback must take 4 char parameters, and it will be called for each set of 4 base-64 characters that are produced (the length of a base-64 string is always a multiple of 4).

Also see choc::base64::encodeToString() for an easier way to get the result as a string.
